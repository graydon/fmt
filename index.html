Welcome to FMT pages.

FMT (Formal Models and Transformations) is a framework for formal models and for transformations between these models.

Download

You can get the source code (master branch) as an archive using the links, above. Please also consider forking the project on GitHub using https://github.com/pascalpoizat/fmt/fork.

Dependencies

The dependencies depend on the models / transformations you are interested in.

For the BPMN model and BPMN->CIF transformations:

EMF framework

required: org.eclipse.emf.common_2.9.1+, org.eclipse.emf.ecore_2.9.1+, org.eclipse.emf.ecore.xmi_2.9.1+.

Available by installing Eclipse https://www.eclipse.org/

BPMN modeller

required: org.eclipse.bpmn2_0.7.0+.

Available using the Eclipse update site http://download.eclipse.org/bpmn2-modeler/updates/.

For the PNML model and CIF->PNML transformations (ongoing work):

PNML framework

required: fr.lip6.move.pnml.framework.coremodel_2.2.1+, fr.lip6.move.pnml.framework.ptnet_2.2.1+, fr.lip6.move.pnml.framework.utils_2.2.1+

Available using the Eclipse update site http://pnml.lip6.fr/pnmlframework/updatesite/ or at http://pnml.lip6.fr/.
