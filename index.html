FMT

Welcome to FMT pages.

FMT (Formal Models and Transformations) is a framework for formal models and for transformations between these models.

Installation

1. Download the source code

You can get the source code (master branch) as an archive using the links, above. Please also consider forking the project on GitHub using https://github.com/pascalpoizat/fmt/fork.

2. Dependencies

2.1 Java

To have FMT running, you will need a Java distribution http://www.java.com/download/

2.2 Model-related dependencies

Depending on the models you are interested in, you will need additional elements.

2.2.a BPMN 2.0 Choreographies (Choreography Diagrams)

To work with BPMN choreographies, you will need the EMF framework and the BPMN model classes.

EMF framework

Required:

  • org.eclipse.emf.common_2.9.1+,
  • org.eclipse.emf.ecore_2.9.1+,
  • org.eclipse.emf.ecore.xmi_2.9.1+.

You can get them by installing Eclipse.

BPMN model classes

Required:

  • org.eclipse.bpmn2_0.7.0+.

You can get it by updating your Eclipse IDE using the BPMN update site.

2.2.b Petri Nets (PNML)

Required:

  • fr.lip6.move.pnml.framework.coremodel_2.2.1+,
  • fr.lip6.move.pnml.framework.ptnet_2.2.1+,
  • fr.lip6.move.pnml.framework.utils_2.2.1+.

You can get them using one of these techniques:

  • update your Eclipse IDE using the PNML framework update site;
  • download the archive of jar files from the PNML framework Web site;
  • retrieve source code from the PNML framework GitHub repository.
2.2.c Choreography Intermediary Language (CIF)

Bundled with FMT.

Compiling the sources

To compile the source and get a jar file, we recommend to use your favorite IDE. Do not forget to add the above-mentioned dependencies in the project libraries.

Typical use

Make a working directory

> mkdir fmt-tests
> cd fmt-tests

Get the sample.bpmn sample BPMN choreography file here and save it in fmt-tests

> ls
sample.bpmn

Run a transformer on the sample file.

> java -cp "$FMT_HOME/fmt.jar:$FMT_DEPENDENCIES/*" transformations.bpmn2cif.Bpmn2Cif sample.bpmn

where:

  • $FMT_HOME is the directory where you have installed the FMT jar file,
  • $FMT_DEPENDENCIESis the directory where you have installed the dependencies.

API main entry points

  • transformations.bpmn2cif.Bpmn2Cif : transforms a BPMN 2.0 choreography into a CIF model
  • transformations.cif2pnml.Cif2Pnml : transforms a CIF model into a PNML model
